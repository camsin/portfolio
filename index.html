DESARROLLO FRONT-END

El desarrollo Front-end consiste en generar los componentes visuales de un sitio web. Utilizando HTML, CSS y Javascript, creo sitios web rápidos e interactivos. Esto también puede incluir el uso de CMS u otras integraciones.

DISEÑO UI/UX

UI/UX implica planificar la estructura y el diseño de un sitio. Una vez que la arquitectura de información adecuada está en su lugar, diseño los componentes visuales para crear una hermosa experiencia de usuario.
